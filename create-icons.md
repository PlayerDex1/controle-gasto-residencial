# Criar Ícones PWA

Para criar os ícones necessários para o PWA, você pode:

1. Criar imagens de 192x192 e 512x512 pixels
2. Salvá-las em `public/icon-192x192.png` e `public/icon-512x512.png`
3. Ou usar um gerador online como https://realfavicongenerator.net/

Por enquanto, você pode usar imagens placeholder ou criar ícones simples.
